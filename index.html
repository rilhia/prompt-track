A track is one JSON sidecar file. It names the video, carries the ground truth, and lists timestamped cues. See sample-track.json in this repository for the format. The video itself is never modified.

Companion

Simulated

The key stays in this page's memory only. It is never stored, logged or sent anywhere except directly to Anthropic from your browser. Refreshing the page forgets it. Without a key the companion runs a scripted simulation of the same flow.

01See one

Lean back mode. The track stays dormant and the video plays clean, start to finish, so you understand the shape of the task before touching a key. Captions for the machine, switched off.

02Do one

Lab mode. At each cue the video pauses itself and the companion hands over the exact command for your machine, from the creator's ground truth, never transcribed off a blurry frame. You do the work, it holds the pace.

03Teach one

The track is a text sidecar in a repository. Where it stumbled for you, you report, the author edits one cue, and the next learner gets a better teacher. No re-recording. The documentation learns.
